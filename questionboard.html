🔧 Firebase Setup

Enable real-time multi-device sync with a free Firebase project — about 5 minutes to set up.

1
Create a Firebase project

Go to console.firebase.google.com → Add project → give it a name → disable Analytics if preferred.

2
Enable Realtime Database

Left sidebar → Build → Realtime Database → Create Database → choose a region → select Test mode.

3
Get your config

Gear icon ⚙ → Project settings → Your apps → click </> → register app → copy the firebaseConfig object.

4
Paste config into this file

Open questionboard.html in a text editor. Find PASTE YOUR FIREBASE CONFIG HERE and replace the placeholder values.

5
Host and share

Upload to GitHub Pages, Netlify, or Firebase Hosting (all free). Use Class URLs in the Teacher tab to get shareable links for each class.
